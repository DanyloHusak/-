Сценарій 1: Бронювання квитка на виставку через онлайн-систему
Name, description: Бронювання квитка через систему.
Actors: Користувач, Система.
Pre-condition: Користувач знаходиться на сторінці бронювання квитків у системі.
Post-condition: Користувач успішно забронював квиток, отримавши підтвердження.

Actor Action: Користувач вибирає опцію "Бронювання квитка".
System Response: Система відображає доступні типи квитків і їх кількість.

Actor Action: Користувач обирає тип квитка і кількість.
System Response: Система перевіряє доступність обраних квитків і відображає повідомлення про доступність або недоступність обраних квитків.
Alternative: Якщо обраний тип квитка недоступний, система пропонує користувачеві вибрати інший тип або іншу кількість квитків.

Actor Action: Користувач вводить персональні дані для бронювання (ім'я, прізвище, контактні дані).
System Response: Система перевіряє коректність введених даних і вимагає заповнення всіх обов'язкових полів.

Actor Action: Користувач вибирає спосіб для онлайн оплати та вводить дані банківської картки.
System Response: Система перевіряє дані картки та відправляє запит на бронювання квитків у систему управління квитками.
Alternative: якщо дані введено некоректно, система повідомляє про помилки та просить користувача внести виправлення.

Actor Action: Користувач підтверджує оплату квитків.
System Response: Система надсилає підтвердження бронювання користувачеві та відображає повідомлення про успішне бронювання.

Scenario: Користувач успішно бронює квиток на виставку через онлайн-систему.


Сценарій 2: Реєстрація нового користувача в системі
Name, description: Реєстрація нового користувача в системі.
Actors: Користувач, Система.
Pre-condition: Користувач знаходиться на сторінці реєстрації.
Post-condition: Новий користувач зареєстрований у системі.

Actor Action: Користувач вибирає опцію "Зареєструватися" на головній сторінці системи.
System Response: Система відображає форму для реєстрації нового користувача.

Actor Action: Користувач вводить свої особисті дані (ім'я, електронну пошту, пароль тощо) у відповідні поля.
System Response: Система перевіряє коректність введених даних і вимагає заповнення всіх обов'язкових полів.
Alternative: якщо дані введено некоректно або деякі обов'язкові поля не заповнені, система відображає відповідне повідомлення і просить виправити помилки.

Actor Action: Після введення даних користувач, натискає кнопку "Зареєструватися".
System Response: Система зберігає дані нового користувача у базі даних і відображає повідомлення про успішну реєстрацію.

Actor Action: Користувач отримує лист на свою електронну пошту з посиланням для підтвердження реєстрації і переходить по посиланню.
System Response: Система підтверджує режстрацію та надає можливість використовувати всі функції системи.

Scenario: Новий користувач успішно реєструється в системі.


Сценарій 3: Оформлення замовлення товару через онлайн-магазин
Name, description: Оформлення замовлення товару через онлайн-магазин.
Actors: Користувач, Система.
Pre-condition: Користувач має товари у кошику і знаходиться на сторінці оформлення замовлення.
Post-condition: Замовлення товару успішно оформлене, користувач отримує підтвердження.

Actor Action: Користувач вибирає товар, який бажає придбати, та натискає кнопку "Додати до кошика".
System Response: Система додає обраний товар до кошика.

Actor Action: Користувач вибирає опцію "Оформити замовлення" і вводить свої контактні дані для доставки.
System Response: Система перевіряє коректність введених даних і вимагає заповнення всіх обов'язкових полів.
Alternative: якщо дані введено некоректно або деякі обов'язкові поля не заповнені, система відображає відповідне повідомлення і просить виправити помилки.

Actor Action: Користувач підтверджує своє замовлення, натискаючи кнопку "Підтвердити замовлення".
System Response: Система зберігає дані замовлення у базі даних і відображає повідомлення про успішне оформлення замовлення.

Scenario: Користувач успішно оформляє замовлення товару через онлайн-магазин.
